Knowledge Base Builder   PyPI   GitHub

Python package for converting diverse content into a search-engine-friendly knowledge base. It effortlessly ingests files (PDFs, DOCXs, spreadsheets), websites, and GitHub repositories, then leverages large language models (Gemini, GPT, Claude) to generate a Markdown knowledge base. Ideal for creating structured and crawlable formats like llms.txt and llms-full.txt, supporting Retrieval-Augmented Generation (RAG) applications, or synthesizing unstructured data from multiple sources. Capable of summarizing an entire github repostory or website in two lines of code.

pip install knowledge-base-builder
